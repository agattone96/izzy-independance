# Layout Definitions (`src/layouts/`)

Layouts provide structural templates that wrap pages across roles and states.

## Target Layouts

- **`PublicLayout`**: Outer container for landing and visitor pages.
- **`AuthLayout`**: Focused screen-centered frame for Login, Registration, and Invite Claims.
- **`ParentLayout`**: Management dashboard layout with controls, navigation rail, and status widgets.
- **`ChildLayout`**: Single-view highly engaging cosmic interface tailored for kid navigation.
- **`CaregiverLayout`**: Caregiver/visit supervisor layout emphasizing parental tasks & bedtime emergency alerts.
- **`DashboardLayout`**: Shared helper abstract layouts containing structural columns and grid presets.
