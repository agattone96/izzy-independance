# Pages (`src/pages/`)

Routed screens organized strictly by user role and functional boundaries. Pages handle layout assembly and hook connections.

## Subdirectories

- **`public/`**: Accessible pages like standard Agreements and rules.
- **`auth/`**: SignIn, Signup, and Joining token gateway forms.
- **`parent/`**: Parents' multi-column control dashboard and logs.
- **`child/`**: Kid-safe task boards and badge cabinets.
- **`caregiver/`**: Babysitter active-visit checklists.
