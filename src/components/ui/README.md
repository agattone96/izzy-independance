# Shared Presentational UI Subcomponents (`src/components/`)

This directory is reserved solely for shared, dumb (presentational) components, organized by utility to keep the workspace clean.

## UI Sub-Categories

- **`ui/`**: Low-level generic layout pieces (cards, standard custom buttons, badges, forms).
- **`navigation/`**: Global navigators, multi-role profile switch tabs.
- **`feedback/`**: Alert boxes, modal popups, interactive confirmation triggers, and loading loaders.
- **`icons/`**: Consistent SVG or Lucide standard mappings.
